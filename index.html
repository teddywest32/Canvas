Canvas comes packed with features, yet keeps a minimal footprint on your web server.

  • edit

    Markdown

    Like writing with markdown shorthand syntax? You'll love the way Canvas stores and displays your content.

  • invert_colors

    Theming

    Custom LESS files are provided so you can modify the theme to your own taste and preference.

  • date_range

    Scheduled Posts

    Write posts and schedule them to publish exactly on the date and time you choose.

  • local_offer

    Tagging

    Canvas allows you to tag posts for categorization, grouping and organization.

  • settings

    Simple Configuration

    A single configuration file holds all of the necessary global variables to set in order to get you up and running in no time.

  • cloud_upload

    Uploading

    Storage and file management come fully functional straight out of the box. Create folders or upload files and images.

About

Canvas is a minimal blogging application for developers. It attempts to make blogging simple and enjoyable by utilizing the latest technologies and keeping the administration as simple as possible with the primary focus on writing.
